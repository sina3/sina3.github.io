3MM

This is a game called Three Men’s Morris. It is a variant of classic game of Tic Tac Toe. Tic Tac Toe is the game that many new developers make using tutorials as beginner projects. I took the concept and worked on a slightly different game logic. I developed the app from ground up so that I get comfortable working independently.

I published the app to App Store as soon as I had a working MVP to get things running. Doing this, I also learned using App Store Connect to setup app information and publish to App Store. I have a lot of features planned that will soon be shipped with the next update.

You can read more about the game on the app website.

Technology Stack

  • Swift and Xcode
  • MVC design pattern
  • UIKit
  • Storyboards

InstaPrep

This is my current project and will be finished soon. InstaPrep’s main purpose is to let user crop big images, specially Panorama pictures, into smaller images fit for use in Instagram and other similar apps.

I am using some of the newer methods introduced by Apple for PhotoKit to work with Photo library. After publishing MVP to App Store, more versatile editing features will be added in later versions.

Technology Stack

  • Swift and Xcode
  • UIKit
  • PhotoKit
  • Swift Package Manager

EmojiArt

Tutorial app as part of CS193P course

EmojiArt was the last and most feature-packed app in the course. It enables dragging images from web browser and importing them in the app, adding custom text and emojis and saving the document for future edits.

Once finished, I was really confident that using these skills, I can recreate a lot of complicated features I see in professional iPhone and iPad apps. In this project, I learned:


  • Drag & Drop
  • UITableView and UICollectionView
  • UITextField and Keyboard
  • Persistence and UIDocument
  • UIDocumentBrowserViewController
  • Using Codable protocol to create a JSON representation of the document
  • Notifications & KVO
  • Application Lifecycle
  • Modal Seugues, Popover, and Unwind Segue
  • Working with camera using UIImagePickerViewController

Technology Stack

  • Swift and Xcode
  • UIKit
  • UIDocuments

Concentration

Tutorial app as part of CS193P course

This is a well-known game where you match pairs of identical cards by trying to remember their location on a grid.

This was my first experience building an iOS app and I learned a lot in the process including:


  • How to work with interface builder
  • Working with UI elements from code using @IBOutlet and @IBAction
  • MVC design pattern
  • ViewController Lifecycle
  • Swift data structures such as structs, enums, optionals, …
  • Automatic Reference Counting and and Memory management
  • Access control
  • How to use extensions to add functionality to data structures
  • Protocols and Delegation
  • Tab Bar Controller, Navigation VC, Split View Controller
  • Segues
  • Size Classes in Autolayout

Technology Stack

  • Swift and Xcode
  • UIKit
  • Storyboards and Autolayout
